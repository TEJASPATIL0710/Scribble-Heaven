# Scribble-Heaven
E-commerce Website

Objective:
The objective of the project is to develop a fully functional e-commerce website. This website should allow users to browse, search for, and purchase products online. It needs to provide a seamless user experience from registration and product browsing to checkout and order tracking. The website should be responsive, secure, and scalable to accommodate growing user traffic and transactions. The overall goal is to create a user-friendly platform that can efficiently handle the entire online shopping process.
